# Git-Font-Hosting
Use Github to host your font files.

## Summary
1. In **font.css** file

   Uses @font-face to define a font
   
   Calls files from **fonts/** folder
   
1. In **example.css** file

   Uses @import to call **font.css** file
   
   Applies font using CSS
